# Project 1 — Machine Learning

This project is part of the **Artificial Intelligence (2EL1580) — Spring 2025** course.

The goal of this assignment is to explore several core concepts in machine learning by implementing different models in PyTorch. Throughout the project, the student builds and trains neural networks for a variety of tasks, ranging from basic binary classification to more advanced sequence modeling and convolutional architectures.

The project begins with the implementation of a **binary perceptron**, then moves on to a **non-linear regression model** to approximate the sine function. It also includes a **digit classification task** on the MNIST dataset, a **language identification model** based on recurrent neural networks, and a **convolutional neural network** for image classification. An extra-credit part introduces **attention mechanisms**, and an optional final section explores a simplified **character-level GPT model**.

The main file to complete is `models.py`, while the rest of the provided code handles datasets, backend utilities, and autograding.

This project serves as an introduction to practical deep learning with PyTorch, with a focus on implementing models, understanding training workflows, and experimenting with architectures for different machine learning problems.
